# Hacking Tool — TryHackMe Python Labs

Python scripts for the TryHackMe Python Labs covering three attack types:

- **Brute Force** — password attacks against a login form
- **Vulnerability Scanner** — detects SQLi and XSS on a target URL
- **Basic Exploit** — sends a reverse shell via command injection

## Requirements

```bash
pip install requests
```

## Usage

```bash
python "Brute_force/bruteforce.py"
python "Brute_force/bruteforce1.py"
python "Vulnerability Scanner/scanner.py"
python "Basic Exploit/exploit.py"
```
