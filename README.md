# hacking_tool_Tryhackme

A small collection of **Python scripts** made for practicing **TryHackMe** web labs (educational use).

## What's inside
- **Brute_force/**: simple login brute-force scripts (`bruteforce.py`, `bruteforce1.py`)
- **Vulnerability Scanner/**: basic SQLi/XSS payload checks (`scanner.py`)
- **Basic Exploit/**: a simple exploit / reverse shell request example (`exploit.py`)

## Requirements
- Python 3
- `requests` library

Install dependencies:
```bash
pip install requests
```

## Usage
Open a script and update the target URL / IP values, then run:
```bash
python3 path/to/script.py
```

## Disclaimer
For **learning and authorized testing only**. Do not use against systems you do not own or have explicit permission to test.
